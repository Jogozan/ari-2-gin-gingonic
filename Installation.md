# Installation de GO

## Pour Linux de la Fac

il n'y a rien a faire GO est déjà installé.

## Pour Linux Personnel

### Téléchargement

1. Télécharger la dernière version de GO
    ```bash
    sudo apt update
    sudo apt install golang-go
    ```

## Pour Windows

### Téléchargement

1. Allez sur golang.org/dl
2. Télécharger les dichier MSI pour Windows
3. Executez le fichier `.msi` téléchargé 

### Installation 

- Suivez l'assistant d'installation
- L'installation par défaut dan `C:\go` est recommandée
- L'installer configure automatiquement les variables d'environnement

## Vérification

Ouvrez un terminal et tapez :

```bash
go version
```

si une erreur comme quoi la commande n'est pas trouvé apparait alors redémarrez le PC.
